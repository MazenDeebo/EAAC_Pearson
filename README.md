# EAAC Pearson BTEC Website

## Project Overview
A modern, responsive website for EAAC Group showcasing Pearson BTEC qualifications in Engineering, Business & Management, IT & Computing, and Artificial Intelligence.

## Design System
Based on EAAC Group's visual identity:
- **Primary Colors**: Deep Navy Blue (#1e3a5f), Bright Red (#e31e24)
- **Typography**: Clean sans-serif fonts
- **Layout**: Centered, responsive grid system
- **Components**: Professional cards, clean navigation, accessible forms

## Technologies Used
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript
- Responsive Design
- Accessibility Features

## Content Focus
- Pearson BTEC Level 5 qualifications
- Engineering programs
- Business & Management courses
- IT & Computing specializations
- Artificial Intelligence programs
- EAAC Group contact and location information

## Development Notes
- Mobile-first responsive design
- SEO optimized structure
- Accessibility compliant (WCAG 2.1)
- Fast loading performance
- Cross-browser compatibility
